# About SPAdes

SPAdes - St. Petersburg genome assembler - is an assembly toolkit containing
various assembly pipelines. This manual will help you to install and run
SPAdes. SPAdes version 3.15.4 was released under GPLv2 on February 1, 2021 and
can be downloaded from <http://cab.spbu.ru/software/spades/>.

The latest SPAdes paper describing various pipelines in a protocol format is
available
[here](https://currentprotocols.onlinelibrary.wiley.com/doi/abs/10.1002/cpbi.102).
