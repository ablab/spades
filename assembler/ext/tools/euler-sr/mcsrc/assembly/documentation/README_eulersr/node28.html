Known issues.

When using mate-pairs to resolve repeats, the sequence of the repeat will be a consensus, so there will likely be many mismatches in repeat regions. The overall accuracy of the layout of the alignment is usually correct though.

Support for running on a rocks cluster has been removed. Since EULER-SR is 100X faster than the first release, running on a cluster is less important. Still, a new release will use multiple cores for error correction.


Mark Chaisson 2008-12-14
